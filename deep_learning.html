Lecture5 では、CIFAR-10 の分類課題に対して、CNN ベースのモデルを PyTorch で実装しています。 今回は API 制限が基本的に緩和されている一方で、CNN ベースであること、torchvision 等の既存モデルや学習済みモデルを使わないことが条件です。 このコードでは、WideResNet 系の CNN を自前で定義し、データ拡張・正則化・EMA・MixUp/CutMix・TTA を組み合わせて、精度向上を狙っています。

1. データ拡張

RandomCrop、RandomHorizontalFlip、RandAugment、RandomErasing を組み合わせ、CIFAR-10 の画像に対して強めの augmentation を適用しています。 これにより、限られた訓練データでも汎化性能を高める構成にしています。

2. モデル構成

モデルは WideResNet-28-10 をベースにした CNN です。 畳み込み層、BatchNorm、ReLU、Dropout、ショートカット接続を組み合わせ、CIFAR-10 向けに高い表現力を持つ構成にしています。

3. 学習の工夫

optimizer には SGD + momentum + Nesterov を使い、CosineAnnealingLR で学習率をスケジューリングしています。 さらに label smoothing、gradient clipping、mixed precision を利用し、安定した学習を狙っています。

4. MixUp / CutMix

学習時には MixUp と CutMix を確率的に切り替え、過学習を抑えながら決定境界を滑らかにしています。 CIFAR-10 のような小規模画像分類では、精度改善に効きやすい工夫です。

5. EMA と TTA

学習中は Exponential Moving Average のモデルを管理し、検証・推論には EMA 側を利用しています。 推論時には original と horizontal flip の平均を取る TTA を行い、提出予測を安定化しています。

6. 全体のポイント

既存モデルや学習済みモデルに頼らず、CNN を自前定義したうえで、WideResNet、強いデータ拡張、MixUp/CutMix、EMA、TTA を組み合わせた構成です。 5月15日時点で 5位 という好成績につながった実装として掲載します。

#!/usr/bin/env python
# coding: utf-8

# # 第5回講義 宿題
# ---
# Lecture 5: Homework

# ## 課題
# 今Lessonで学んだことに工夫を加えて，CNNでより高精度なCIFAR10の分類器を実装してみましょう．
# ---
# Assignment
# Based on what you learned in this lesson, let's implement a more accurate CIFAR-10 classifier using a CNN by adding your own improvements.

# ### 目標値
# Accuracy 76%

# ### ルール
# - 訓練データは`x_train`， `t_train`，テストデータは`x_test`で与えられます．
# - 予測ラベルは one_hot表現ではなく0~9のクラスラベル で表してください．
# - 下のセルで指定されている`x_train`，`t_train`以外の学習データは使わないでください．
# - Pytorchを利用して構いません．
# - 今回から基本的にAPI制限はありません．
# - ただしCNNベースでないモデル（Vision Transformerなど）やtorchvision等の既存モデル，学習済みモデルは用いないでください．

from google.colab import drive
drive.mount('/content/drive')

work_dir = 'drive/MyDrive/Colab Notebooks/DLBasics2026_colab'

import os
import copy
import random
import numpy as np
import pandas as pd
from PIL import Image
from sklearn.model_selection import train_test_split

import torch
import torch.nn as nn
import torch.nn.functional as F
import torch.optim as optim
from torchvision import transforms
from torch.utils.data import Dataset, DataLoader, Subset

x_train = np.load(work_dir + '/Lecture05/data/x_train.npy')
t_train = np.load(work_dir + '/Lecture05/data/t_train.npy')
x_test = np.load(work_dir + '/Lecture05/data/x_test.npy')

# -----------------------------
# Reproducibility / device
# -----------------------------
def fix_seed(seed=42):
    random.seed(seed)
    np.random.seed(seed)
    torch.manual_seed(seed)
    torch.cuda.manual_seed_all(seed)

fix_seed(42)

device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
torch.backends.cudnn.benchmark = True

# -----------------------------
# Dataset with independent transforms for train/valid/test
# -----------------------------
class CIFAR10ArrayDataset(Dataset):
    def __init__(self, x, t=None, transform=None):
        self.x = x.astype(np.uint8)
        self.t = None if t is None else t.astype(np.int64)
        self.transform = transform

    def __len__(self):
        return len(self.x)

    def __getitem__(self, idx):
        img = Image.fromarray(self.x[idx])
        if self.transform is not None:
            img = self.transform(img)
        if self.t is None:
            return img
        return img, torch.tensor(self.t[idx], dtype=torch.long)

# Use statistics computed only from the given training data.
_x = x_train.astype(np.float32) / 255.0
mean = _x.mean(axis=(0, 1, 2)).tolist()
std = _x.std(axis=(0, 1, 2)).tolist()
print("mean:", mean, "std:", std)

# Strong augmentation. RandAugment is allowed because it is not a pretrained/existing model.
train_tfms = [
    transforms.RandomCrop(32, padding=4, padding_mode="reflect"),
    transforms.RandomHorizontalFlip(p=0.5),
]
try:
    train_tfms.append(transforms.RandAugment(num_ops=2, magnitude=9))
except Exception:
    # Fallback for older torchvision versions.
    train_tfms += [
        transforms.ColorJitter(brightness=0.2, contrast=0.2, saturation=0.2, hue=0.05),
        transforms.RandomAffine(degrees=10, translate=(0.08, 0.08)),
    ]
train_tfms += [
    transforms.ToTensor(),
    transforms.Normalize(mean, std),
    transforms.RandomErasing(p=0.25, scale=(0.02, 0.15), ratio=(0.3, 3.3), value="random"),
]
transform_train = transforms.Compose(train_tfms)
transform_test = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize(mean, std),
])

# Stratified validation split. Validation is NOT augmented.
val_size = 5000
train_idx, val_idx = train_test_split(
    np.arange(len(x_train)),
    test_size=val_size,
    stratify=t_train,
    random_state=42,
)

train_dataset = CIFAR10ArrayDataset(x_train, t_train, transform_train)
valid_dataset = CIFAR10ArrayDataset(x_train, t_train, transform_test)
test_dataset  = CIFAR10ArrayDataset(x_test, None, transform_test)

train_data = Subset(train_dataset, train_idx)
val_data   = Subset(valid_dataset, val_idx)
test_data  = test_dataset

batch_size = 128
num_workers = 2
pin_memory = (device.type == "cuda")

dataloader_train = DataLoader(train_data, batch_size=batch_size, shuffle=True,
                              num_workers=num_workers, pin_memory=pin_memory, drop_last=True)
dataloader_valid = DataLoader(val_data, batch_size=batch_size, shuffle=False,
                              num_workers=num_workers, pin_memory=pin_memory)
dataloader_test = DataLoader(test_data, batch_size=batch_size, shuffle=False,
                             num_workers=num_workers, pin_memory=pin_memory)

# -----------------------------
# Custom WideResNet for CIFAR-10
# CNN-based, trained from scratch, no torchvision model/pretraining.
# -----------------------------
class WideBasicBlock(nn.Module):
    def __init__(self, in_planes, out_planes, stride, dropout_rate):
        super().__init__()
        self.bn1 = nn.BatchNorm2d(in_planes)
        self.conv1 = nn.Conv2d(in_planes, out_planes, kernel_size=3, padding=1, bias=False)
        self.bn2 = nn.BatchNorm2d(out_planes)
        self.conv2 = nn.Conv2d(out_planes, out_planes, kernel_size=3, stride=stride, padding=1, bias=False)
        self.dropout_rate = dropout_rate
        self.shortcut = nn.Identity()
        if stride != 1 or in_planes != out_planes:
            self.shortcut = nn.Conv2d(in_planes, out_planes, kernel_size=1, stride=stride, bias=False)

    def forward(self, x):
        out = F.relu(self.bn1(x), inplace=True)
        shortcut = self.shortcut(out if not isinstance(self.shortcut, nn.Identity) else x)
        out = self.conv1(out)
        out = F.relu(self.bn2(out), inplace=True)
        if self.dropout_rate > 0:
            out = F.dropout(out, p=self.dropout_rate, training=self.training)
        out = self.conv2(out)
        return out + shortcut

class WideResNet(nn.Module):
    def __init__(self, depth=28, widen_factor=10, dropout_rate=0.3, num_classes=10):
        super().__init__()
        assert (depth - 4) % 6 == 0
        n = (depth - 4) // 6
        channels = [16, 16 * widen_factor, 32 * widen_factor, 64 * widen_factor]
        self.in_planes = channels[0]
        self.conv1 = nn.Conv2d(3, channels[0], kernel_size=3, padding=1, bias=False)
        self.layer1 = self._make_layer(channels[1], n, stride=1, dropout_rate=dropout_rate)
        self.layer2 = self._make_layer(channels[2], n, stride=2, dropout_rate=dropout_rate)
        self.layer3 = self._make_layer(channels[3], n, stride=2, dropout_rate=dropout_rate)
        self.bn = nn.BatchNorm2d(channels[3])
        self.fc = nn.Linear(channels[3], num_classes)

    def _make_layer(self, out_planes, num_blocks, stride, dropout_rate):
        strides = [stride] + [1] * (num_blocks - 1)
        layers = []
        for s in strides:
            layers.append(WideBasicBlock(self.in_planes, out_planes, s, dropout_rate))
            self.in_planes = out_planes
        return nn.Sequential(*layers)

    def forward(self, x):
        out = self.conv1(x)
        out = self.layer1(out)
        out = self.layer2(out)
        out = self.layer3(out)
        out = F.relu(self.bn(out), inplace=True)
        out = F.adaptive_avg_pool2d(out, 1).flatten(1)
        return self.fc(out)

def init_weights(m):
    if isinstance(m, nn.Conv2d):
        nn.init.kaiming_normal_(m.weight, mode="fan_out", nonlinearity="relu")
    elif isinstance(m, nn.Linear):
        nn.init.kaiming_normal_(m.weight)
        if m.bias is not None:
            nn.init.zeros_(m.bias)
    elif isinstance(m, nn.BatchNorm2d):
        nn.init.ones_(m.weight)
        nn.init.zeros_(m.bias)

conv_net = WideResNet(depth=28, widen_factor=10, dropout_rate=0.3, num_classes=10).to(device)
conv_net.apply(init_weights)

# Exponential Moving Average usually improves validation/test accuracy.
ema_net = copy.deepcopy(conv_net).to(device)
ema_net.eval()
for p in ema_net.parameters():
    p.requires_grad_(False)

@torch.no_grad()
def update_ema(model, ema_model, decay=0.999):
    msd = model.state_dict()
    for k, v in ema_model.state_dict().items():
        if v.dtype.is_floating_point:
            v.mul_(decay).add_(msd[k].detach(), alpha=1.0 - decay)
        else:
            v.copy_(msd[k])

# -----------------------------
# MixUp / CutMix
# -----------------------------
def rand_bbox(size, lam):
    W = size[3]
    H = size[2]
    cut_rat = np.sqrt(1.0 - lam)
    cut_w = int(W * cut_rat)
    cut_h = int(H * cut_rat)
    cx = np.random.randint(W)
    cy = np.random.randint(H)
    bbx1 = np.clip(cx - cut_w // 2, 0, W)
    bby1 = np.clip(cy - cut_h // 2, 0, H)
    bbx2 = np.clip(cx + cut_w // 2, 0, W)
    bby2 = np.clip(cy + cut_h // 2, 0, H)
    return bbx1, bby1, bbx2, bby2

def ce_loss(logits, targets):
    return F.cross_entropy(logits, targets, label_smoothing=0.1)

def mixed_criterion(logits, y_a, y_b, lam):
    return lam * ce_loss(logits, y_a) + (1.0 - lam) * ce_loss(logits, y_b)

# -----------------------------
# Training setup
# -----------------------------
n_epochs = 200
lr = 0.1
optimizer = optim.SGD(conv_net.parameters(), lr=lr, momentum=0.9, weight_decay=5e-4, nesterov=True)
scheduler = optim.lr_scheduler.CosineAnnealingLR(optimizer, T_max=n_epochs)
scaler = torch.cuda.amp.GradScaler(enabled=(device.type == "cuda"))

mixup_alpha = 0.8
cutmix_alpha = 1.0
cutmix_prob = 0.5
best_acc = 0.0
best_state = None

# -----------------------------
# Train / validate
# -----------------------------
for epoch in range(1, n_epochs + 1):
    conv_net.train()
    losses_train = []
    n_train = 0
    acc_train = 0

    for x, t in dataloader_train:
        x = x.to(device, non_blocking=True)
        t = t.to(device, non_blocking=True)
        optimizer.zero_grad(set_to_none=True)

        r = np.random.rand()
        if r < cutmix_prob:
            lam = np.random.beta(cutmix_alpha, cutmix_alpha)
            rand_index = torch.randperm(x.size(0), device=device)
            y_a, y_b = t, t[rand_index]
            bbx1, bby1, bbx2, bby2 = rand_bbox(x.size(), lam)
            x[:, :, bby1:bby2, bbx1:bbx2] = x[rand_index, :, bby1:bby2, bbx1:bbx2]
            lam = 1.0 - ((bbx2 - bbx1) * (bby2 - bby1) / (x.size(-1) * x.size(-2)))
            with torch.cuda.amp.autocast(enabled=(device.type == "cuda")):
                y = conv_net(x)
                loss = mixed_criterion(y, y_a, y_b, lam)
        else:
            lam = np.random.beta(mixup_alpha, mixup_alpha)
            rand_index = torch.randperm(x.size(0), device=device)
            y_a, y_b = t, t[rand_index]
            mixed_x = lam * x + (1.0 - lam) * x[rand_index]
            with torch.cuda.amp.autocast(enabled=(device.type == "cuda")):
                y = conv_net(mixed_x)
                loss = mixed_criterion(y, y_a, y_b, lam)

        scaler.scale(loss).backward()
        scaler.unscale_(optimizer)
        torch.nn.utils.clip_grad_norm_(conv_net.parameters(), max_norm=5.0)
        scaler.step(optimizer)
        scaler.update()
        update_ema(conv_net, ema_net, decay=0.999)

        pred = y.argmax(dim=1)
        acc_train += (pred == t).float().sum().item()
        n_train += t.size(0)
        losses_train.append(loss.item())

    scheduler.step()

    # Validate with EMA model and deterministic transform.
    ema_net.eval()
    losses_valid = []
    n_val = 0
    acc_val = 0
    with torch.no_grad():
        for x, t in dataloader_valid:
            x = x.to(device, non_blocking=True)
            t = t.to(device, non_blocking=True)
            y = ema_net(x)
            loss = ce_loss(y, t)
            pred = y.argmax(dim=1)
            acc_val += (pred == t).float().sum().item()
            n_val += t.size(0)
            losses_valid.append(loss.item())

    val_acc = acc_val / n_val
    if val_acc > best_acc:
        best_acc = val_acc
        best_state = copy.deepcopy(ema_net.state_dict())
        torch.save(best_state, work_dir + "/Lecture05/best_wrn28_10_ema.pth")

    print(
        f"EPOCH: {epoch:03d}, "
        f"Train [Loss: {np.mean(losses_train):.4f}, Accuracy(raw): {acc_train/n_train:.4f}], "
        f"Valid [Loss: {np.mean(losses_valid):.4f}, Accuracy: {val_acc:.4f}], "
        f"Best: {best_acc:.4f}"
    )

# -----------------------------
# Prediction with TTA: original + horizontal flip
# -----------------------------
if best_state is not None:
    ema_net.load_state_dict(best_state)
ema_net.eval()

t_pred = []
with torch.no_grad():
    for x in dataloader_test:
        x = x.to(device, non_blocking=True)
        y1 = ema_net(x)
        y2 = ema_net(torch.flip(x, dims=[3]))
        y = (y1 + y2) / 2.0
        pred = y.argmax(dim=1).cpu().tolist()
        t_pred.extend(pred)

submission = pd.Series(t_pred, name="label")
submission.to_csv(work_dir + "/Lecture05/submission_pred_05.csv", header=True, index_label="id")
print("saved:", work_dir + "/Lecture05/submission_pred_05.csv")
print("best validation accuracy:", best_acc)
